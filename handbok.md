## Github cmd:

1. Fyrir nýjustu uppfærslur

```bash
  git pull
```

2. Bæta skjalinu við á git

```bash
  git add {Filename}
```

3. Ready for send

```bash
  git commit -m "{message}"
```

4. Send

```bash
  git push
```

## Github Desktop:

https://desktop.github.com/download/

1. Logga sig fyrst inn
2. Clonea verkefnið (skvis)
3. pulla seinustu uppfærslur (Mikilvægt)
   (Valfrjálst VVV)
4. Opna skjalið í VSCode
5. Breyta kóða/texta
6. Commita í Github Desktop og bæta við skilaboðum
7. Pusha uppfærslurnar þínar
